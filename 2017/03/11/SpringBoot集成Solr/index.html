Apache Solr

下载

  • Apache Solr链接地址：http://www-eu.apache.org/dist/lucene/solr/6.4.2/
  • 选择solr-6.4.2.zip

    安装

  • 解压solr-6.4.2.zip
  • 找到安装的位置，在bin目录中, shift + 鼠标右键选择在此处打开命令窗口
  • 输入solr start
  • 访问路径：http://localhost:8983/solr
