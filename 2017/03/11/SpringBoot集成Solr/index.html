Apache Solr

下载

  • Apache Solr链接地址：http://www-eu.apache.org/dist/lucene/solr/6.4.2/
  • 选择solr-6.4.2.zip

    安装

  • 解压solr-6.4.2.zip
  • 找到安装的位置，在bin目录下, shift + 鼠标右键选择在此处打开命令窗口
  • 输入solr start，控制台显示如下语句表明开启成功，默认端口8983。

Solr视图

访问路径：http://localhost:8983/solr

索引数据

建立Core

  • 创建配置目录demo_config与demo核心
  • 在bin目录下输入solr create -c demo -d demo_config，显示如下语句表明创建成功。

Core视图

访问路径：http://localhost:8983/solr

修改Schema

6.4版本没有schema.xml，只有managed-shcema。打开Core视图，点击Schema，如图。

索引数据

  • 定位目录到solr/example/exampledocs下
  • 在控制台输入java -Dtype=text/csv -Durl=http://localhost:8983/solr/demo/update -jar post.jar books.csv

访问索引文档

访问路径：http://localhost:8983/solr/demo/select?q=name:"C“
